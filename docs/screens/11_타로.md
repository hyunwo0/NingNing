### 화면 구성

#### 1) 타이틀
- "타로 한 장" (2xl, bold)
- "궁금한 주제를 고르고 카드를 한 장 뽑아보세요"

#### 2) 질문 유형 선택 (3개)
- **연애**: 사랑/관계에 대한 질문
- **진로**: 일/커리어에 대한 질문
- **오늘**: 오늘 하루 전반에 대한 질문
- 스타일: 선택된 항목은 bg-foreground text-background

#### 3) 카드 뽑기
- 뒷면 카드 3장 표시 (호버 시 살짝 들어올림)
- 클릭하면 해당 카드 선택 (앞면으로 뒤집는 애니메이션)
- 카드는 실제 타로 78장 중 랜덤 1장 배정

#### 4) CTA
- 질문 유형 + 카드 모두 선택 시 활성화
- "타로 해석 보기" → `/loading-screen?type=tarot`

### 데이터 저장
- `sessionStorage.setItem('tarotInput', ...)` — 질문 유형 + 선택된 카드 정보
- 기존 타로 캐시 초기화: `tarotResult` 삭제

## 공유
- **타로 공유 카드**: AI 이미지 + 질문 유형 + 카드 이름 + 키워드 태그 + advice
- 모바일: OS 공유 시트 / PC: 이미지 다운로드

## API

- `POST /api/tarot` — 타로 카드 + 질문 유형 → AI 해석
- Body: `{ card: { name, keywords }, questionType: 'love' | 'career' | 'today' }`
- Response: `{ interpretation, advice, oneLiner }`

## 타로 카드 데이터
- 메이저 아르카나 22장으로 시작 (추후 78장 확장)
- 각 카드: name(영어), nameKo(한국어), keywords(한국어 3개)
