# 07. 심층 리포트 페이지

> 경로: `/report` | 파일: `src/app/report/page.tsx` | 렌더링: Client Component

## 역할

유료 심층 리포트를 표시하는 페이지. `/result`에서 "심층 리포트 보기" 버튼을 통해 진입합니다.

## 진입 조건

- sessionStorage에 `sajuResult`와 `sajuInput`이 필요
- 없으면 `/input`으로 리다이렉트

## 동작 흐름

1. sessionStorage에서 사주 결과 + 입력 데이터 읽기
2. `POST /api/report` → 심층 리포트 수신
3. 섹션별 카드로 표시

## 화면 구성

### 1) GNB
- title: "심층 리포트"

### 2) 나의 에너지 프로필
- 검은 배경 카드
- 프로필 타이틀 (xl, bold) + 설명 텍스트

### 3) 오늘의 상세 해석
- 전체 흐름 / 연애 / 일/직장 / 재물
- 각 항목 border-t로 구분

### 4) 오늘의 행동 조언
- 3가지 조언, 번호 뱃지 + 텍스트

### 5) 이번 주 흐름
- 2~3문장

### 6) 마무리
- 검은 배경 카드, 중앙 정렬

### 7) 하단 버튼
- "결과 화면으로 돌아가기" → `/result`

## 로딩
- `/loading-screen?type=report`에서 처리

## 에러 화면
- "오류가 발생했습니다" + 에러 메시지 + "결과 화면으로 돌아가기" 버튼

## 캐시 정책
- 리포트 결과를 `sessionStorage`(`sajuReport`)에 캐싱
- 재진입 시 캐시 있으면 즉시 표시
- `/input`에서 새로 제출하면 캐시 삭제

## API 호출
- `POST /api/report` — `/loading?type=report`에서 호출, 이 페이지는 캐시 읽기만 수행
- Body: analysis (fourPillars, fiveElements, dayMaster, dayMasterElement, dayMasterStrength), daily, gender
- Response: `{ report: ReportData }`
