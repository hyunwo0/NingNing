# 10. 개인정보처리방침 페이지

> 경로: `/privacy` | 파일: `src/app/privacy/page.tsx` | 렌더링: Server Component

## 역할

NingNing 서비스의 개인정보 처리방침을 표시하는 정적 페이지입니다.

## 화면 구성

### 1) 뒤로가기 링크
- "← 홈으로 돌아가기" → `/`

### 2) 페이지 헤더
- **제목**: "개인정보 처리방침" (xl, bold)
- **시행일**: "시행일: 2026년 4월 4일"

### 3) 방침 본문 (9개 조항)

| # | 제목 | 주요 내용 |
|---|------|-----------|
| 1 | 총칙 | 개인정보 보호법 준수, 보호 조치 안내 |
| 2 | 수집하는 개인정보 | 생년월일, 출생시간, 성별 |
| 3 | 수집 방법 | 사주 입력 폼을 통해 직접 입력 |
| 4 | 이용 목적 | 사주 팔자 계산, AI 운세 해석 제공 |
| 5 | 보유 기간 | 세션 기반 처리 (sessionStorage), 서버 미저장, 탭 닫으면 자동 삭제 |
| 6 | 제3자 제공 | Anthropic Claude API에 비식별 형태로 전달 |
| 7 | 이용자 권리 | 브라우저 데이터 직접 삭제 가능, 별도 삭제 요청 불필요 |
| 8 | 쿠키 사용 | 필수 쿠키만 사용, 마케팅/분석 쿠키 미사용 |
| 9 | 문의 | [운영자 정보 추후 기재] |
