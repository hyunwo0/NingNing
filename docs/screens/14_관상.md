## 공유
- **관상 공유 카드**: AI 이미지 + 첫인상/성격/매력 각 첫 문장만 표시
- 모바일: OS 공유 시트 / PC: 이미지 다운로드

## API

- `POST /api/face` — base64 이미지 → Claude Vision → AI 관상 분석
- Body: `{ image: string (base64) }`
- Response: `{ title, firstImpression, personality, charm, oneLiner }`
- 이미지는 서버에 저장하지 않음
