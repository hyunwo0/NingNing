# 04. 생년월일시 입력 페이지

> 경로: `/input` | 파일: `src/app/input/page.tsx` | 렌더링: Client Component

## 역할

사용자가 생년월일시와 성별을 입력하는 폼 페이지. 입력 완료 시 데이터를 sessionStorage에 저장하고 `/result`로 이동합니다.

## 화면 구성

### 1) 페이지 헤더
- **제목**: "생년월일시 입력" (2xl, bold)
- **설명**: "정확한 사주 분석을 위해 출생 정보를 입력해주세요"

### 2) 양력/음력 선택
- **라벨**: "달력 유형"
- **버튼 2개**: 양력 | 음력 (grid 2열)
  - 선택됨: bg-foreground text-background + border-foreground
  - 미선택: bg-background text-foreground + border-border
- **윤달 체크박스**: 음력 선택 시에만 노출
  - "윤달" 라벨 + checkbox

### 3) 생년월일 선택
- **라벨**: "생년월일"
- **드롭다운 3개**: 연 | 월 | 일 (grid 3열)
  - 연도: 2010 ~ 1940 (내림차순), 기본값 1995
  - 월: 1 ~ 12
  - 일: 연/월에 따라 동적 (양력: 정확한 일수, 음력: 최대 30일)
- **자동 조정**: 선택된 일이 최대 일수 초과 시 자동 보정
- **스타일**: h-11, appearance-none, 우측 ChevronDown 아이콘

### 4) 태어난 시간 선택
- **라벨**: "태어난 시간"
- **드롭다운 1개**: "모름" + 12시진 (자시~해시)
  - 각 옵션: `시진명 (시간 범위)` 형식
  - 기본값: "모름"
- **보조 문구**: "정확한 시간을 모르면 \"모름\"을 선택하세요. 시주 없이 분석합니다."

### 5) 성별 선택
- **라벨**: "성별"
- **버튼 2개**: 남성 | 여성 (grid 2열)
  - 스타일: 양력/음력 버튼과 동일

### 6) 제출 버튼
- **텍스트**: "오늘의 운세 보기" / "분석 중..."
- **스타일**: h-12, w-full, rounded-xl (Button 컴포넌트, size="lg")
- **비활성화 조건**: 성별 미선택 또는 제출 중

### 7) 하단 안내
- "입력하신 정보는 사주 분석에만 사용되며, / 별도 동의 없이 저장되지 않습니다."

## 상태 관리
- `calendarType`: 'solar' | 'lunar'
- `isLeapMonth`: boolean (윤달 여부)
- `birthYear`: number (기본값 1995)
- `birthMonth`: number (기본값 1)
- `birthDay`: number (기본값 1)
- `birthTime`: string (기본값 'unknown')
- `gender`: 'male' | 'female' | null
- `isSubmitting`: boolean

## 데이터 저장
- 제출 시 `sessionStorage.setItem('sajuInput', ...)` 저장
- 저장 항목: calendarType, isLeapMonth, birthYear, birthMonth, birthDay, birthTime, gender

## 이동 경로

| 요소 | 이동 대상 |
|------|-----------|
| 운세 보기 버튼 | `/result` |
