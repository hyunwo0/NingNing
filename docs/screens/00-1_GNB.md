# 00-1. GNB (Global Navigation Bar)

> 컴포넌트: `src/components/layout/GNB.tsx`

## 규칙

| | 좌측 | 우측 |
|---|------|------|
| 메인(`/`) | NingNing 로고 (클릭 비활성) | AuthNavLink (로그인/마이페이지) |
| 나머지 전체 | ← 뒤로가기 (이전 화면, 없으면 홈) | AuthNavLink (로그인/마이페이지) |

## Props

```typescript
interface GNBProps {
  isHome?: boolean; // true면 로고, false면 뒤로가기
}
```

## 레이아웃

```
메인:
┌─────────────────────────────────┐
│  NingNing              [로그인] │
└─────────────────────────────────┘

하위 페이지:
┌─────────────────────────────────┐
│  ←                     [로그인] │
└─────────────────────────────────┘
```

## 적용 페이지

모든 페이지에 적용. 페이지별 개별 네비게이션/헤더는 삭제.
