# 00-1. GNB (Global Navigation Bar)

> 컴포넌트: `src/components/layout/GNB.tsx`

## 규칙

| | 좌측 | 중앙 | 우측 |
|---|------|------|------|
| 메인(`/`) | NingNing 로고 (클릭 비활성) | 없음 | 마이페이지 아이콘 (또는 로그인) |
| 나머지 전체 | ← 뒤로가기 | 페이지 이름 | 홈 아이콘 + 마이페이지 아이콘 (또는 로그인) |

## Props

```typescript
interface GNBProps {
  isHome?: boolean; // true면 로고, false면 뒤로가기
  title?: string;   // 중앙에 표시할 페이지 이름
}
```

## 레이아웃

```
메인:
┌─────────────────────────────────┐
│  NingNing                  [👤] │
└─────────────────────────────────┘

하위 페이지:
┌─────────────────────────────────┐
│  ←      페이지이름      [🏠][👤] │
└─────────────────────────────────┘
```

## 스타일
- 높이: `h-14` 고정
- 좌측: `w-10` 고정
- 우측: `items-end`로 아이콘 하단 정렬
- 홈 아이콘: size-6, 집 모양
- 마이페이지 아이콘: size-6, 사람 모양 (라벨 없음)
- 로그인: "로그인" 텍스트 링크
- 제목: text-base, font-semibold

## 페이지별 title

| 페이지 | title |
|--------|-------|
| `/` (메인) | 없음 |
| `/input` | 오늘의 운세 |
| `/result` | 오늘의 운세 |
| `/question` | AI 추가질문 |
| `/report` | 심층 리포트 |
| `/login` | 로그인 |
| `/signup` | 회원가입 |
| `/my` | 마이페이지 |
| `/tarot`, `/tarot/result` | 타로 |
| `/mbti`, `/mbti/result` | MBTI |
| `/compatibility`, `/compatibility/result` | 궁합 |
| `/face`, `/face/result` | 관상 |

## 적용 페이지

모든 페이지에 적용. 페이지별 개별 네비게이션/헤더는 삭제.
