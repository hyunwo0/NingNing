# 13. 나와 맞는 연예인

## 페이지 구성

| 경로 | 역할 | 파일 |
|------|------|------|
| `/celeb` | 생년월일 입력 (사주 input 재활용) | `src/app/celeb/page.tsx` |
| `/loading-screen?type=celeb` | 로딩 | 기존 loading-screen 확장 |
| `/celeb/result` | 연예인 매칭 결과 | `src/app/celeb/result/page.tsx` |

## /celeb — 입력

- 기존 `/input`과 동일한 생년월일시 입력
- 또는 이미 사주 입력이 있으면 바로 로딩으로 이동

## API

- `POST /api/celeb` — 생년월일시 + 성별 → AI가 궁합 맞는 연예인 3명 매칭
- Response: `{ matches: [{ name, reason, compatibility }] }`
