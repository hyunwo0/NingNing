# 01. 메인 랜딩 페이지

> 경로: `/` | 파일: `src/app/page.tsx` | 렌더링: Server Component

## 역할

서비스의 첫 화면. 사용자가 "오늘의 운세 보기" 버튼을 눌러 `/input`으로 이동하도록 유도합니다.

## 화면 구성

### 1) 상단 네비게이션
- **위치**: 우측 상단
- **컴포넌트**: `AuthNavLink` (Client Component)
- **동작**:
  - 비로그인 → "로그인" 텍스트 링크 → `/login`
  - 로그인 → 사용자 아이콘 + "마이페이지" → `/my`

### 2) 히어로 영역
- **로고**: 검은 배경 사각형(16x16) 안에 "N" 글자
- **서비스명**: "NingNing" (3xl, bold)
- **한 줄 소개**: "사주명리학 기반 AI 오늘의 운세"
- **설명 문구**: "정통 사주 계산과 AI 해석으로 / 오늘 하루를 위한 현실적인 가이드를 제공합니다"
- **CTA 버튼**: "오늘의 운세 보기" → `/input`
  - 스타일: h-12, rounded-xl, bg-foreground
- **보조 문구**: "회원가입 없이 바로 이용할 수 있습니다"

### 3) 특징 소개 (3개)
| 아이콘 | 제목 | 설명 |
|--------|------|------|
| CompassIcon | 정통 사주 계산 | 입춘/절기 기준의 정확한 사주 팔자 계산... |
| SparklesIcon | AI 맞춤 해석 | 사주 구조를 근거로 연애, 일, 재물 3가지 축... |
| ShieldIcon | 안전한 운세 | 공포 조장이나 확정적 예언 없이... |

- 레이아웃: 아이콘(10x10 muted 배경) + 텍스트 가로 배치

### 4) 이용 방법 (3단계)
| 번호 | 제목 | 설명 |
|------|------|------|
| 1 | 생년월일시 입력 | 양력 또는 음력, 태어난 시간을 선택하세요 |
| 2 | 사주 분석 | 사주 팔자와 오행 분포를 계산합니다 |
| 3 | AI 운세 확인 | 오늘의 연애, 일, 재물 운세를 확인하세요 |

- 레이아웃: 번호 원형 뱃지(8x8) + 텍스트

### 5) 하단 CTA
- **버튼**: "무료로 시작하기" → `/input`
  - 스타일: h-12, w-full, rounded-xl

### 6) 테마별 랜딩 링크
- "연애운이 궁금하다면? →" → `/landing/love`
- "이직을 고민 중이라면? →" → `/landing/career`

### 7) 푸터
- "NingNing · AI 운세 서비스"
- "사주 결과는 참고용이며, 중요한 결정의 근거로 사용하지 마세요."
- 링크: 개인정보처리방침(`/privacy`), 이용약관(`/terms`)

## 이동 경로

| 요소 | 이동 대상 |
|------|-----------|
| CTA 버튼 (x2) | `/input` |
| 로그인 링크 | `/login` |
| 마이페이지 링크 | `/my` |
| 연애운 링크 | `/landing/love` |
| 이직운 링크 | `/landing/career` |
| 개인정보처리방침 | `/privacy` |
| 이용약관 | `/terms` |

- 섹션 구분: 가로 12px 선 (`w-12 h-px bg-border`)
