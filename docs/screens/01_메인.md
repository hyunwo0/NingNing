# 01. 메인 랜딩 페이지

> 경로: `/` | 파일: `src/app/page.tsx` | 렌더링: Client Component

## 역할

서비스의 첫 화면. 다양한 운세 타입 중 하나를 선택해 시작할 수 있는 허브 페이지입니다.

## 레이아웃

```
┌─────────────────────────────────┐
│  NingNing    [로그인/마이페이지] │ ← GNB 헤더
├─────────────────────────────────┤
│                                 │
│   오늘, 뭐 볼까?               │ ← 타이틀
│                                 │
│   ┌─────────────────────────┐   │
│   │  🔮 오늘의 운세           │   │
│   │  사주로 보는 오늘의 흐름  │   │ ← → /input
│   └─────────────────────────┘   │
│                                 │
│   ┌─────────────────────────┐   │
│   │  🃏 타로 한 장            │   │
│   │  카드가 말해주는 오늘     │   │ ← → /tarot
│   └─────────────────────────┘   │
│                                 │
│   ┌─────────────────────────┐   │
│   │  🧠 MBTI 운세            │   │
│   │  내 유형에 맞는 오늘     │   │ ← → /mbti (준비 중)
│   └─────────────────────────┘   │
│                                 │
│   ┌─────────────────────────┐   │
│   │  💕 궁합 보기              │   │
│   │  두 사람의 사주 궁합 분석 │   │ ← → /compatibility
│   └─────────────────────────┘   │
│                                 │
│   ┌─────────────────────────┐   │
│   │  👀 AI 관상              │   │
│   │  얼굴로 보는 나의 성격   │   │ ← → /face (준비 중)
│   └─────────────────────────┘   │
│                                 │
│ ─────────────────────────────── │
│   NingNing · AI 운세 서비스     │
│   사주 결과는 참고용이며...     │
│   개인정보처리방침 · 이용약관   │
└─────────────────────────────────┘
```

## 화면 구성

### 1) GNB 헤더
- **좌측**: "NingNing" 로고 텍스트 (bold) → `/` (홈 링크)
- **우측**: `AuthNavLink` (Client Component)
  - 비로그인 → "로그인" 텍스트 링크 → `/login`
  - 로그인 → 사용자 아이콘 + "마이페이지" → `/my`

### 2) 타이틀
- "오늘, 뭐 볼까?" (2xl, bold)

### 3) 타입 선택 카드
- 각 카드: 라운드 카드(rounded-2xl, border), 클릭 시 해당 페이지로 이동
- 카드 구성: 아이콘 + 타이틀(sm, semibold) + 설명(xs, muted)
- 미구현 타입은 "준비 중" 뱃지 + 클릭 비활성화

| 타입 | 타이틀 | 설명 | 이동 | 상태 |
|------|--------|------|------|------|
| 🔮 | 오늘의 운세 | 사주로 보는 오늘의 흐름 | `/input` | 활성 |
| 🃏 | 타로 한 장 | 카드가 말해주는 오늘 | `/tarot` | 활성 (구현 후) |
| 🧠 | MBTI 운세 | 내 유형에 맞는 오늘 | `/mbti` | 준비 중 |
| 💕 | 궁합 보기 | 두 사람의 사주 궁합 분석 | `/compatibility` | 활성 |
| 👀 | AI 관상 | 얼굴로 보는 나의 성격 | `/face` | 준비 중 |

### 4) 푸터
- "NingNing · AI 운세 서비스"
- "사주 결과는 참고용이며, 중요한 결정의 근거로 사용하지 마세요."
- 링크: 개인정보처리방침(`/privacy`), 이용약관(`/terms`)

## 이동 경로

| 요소 | 이동 대상 |
|------|-----------|
| 오늘의 운세 카드 | `/input` |
| 타로 한 장 카드 | `/tarot` |
| 로그인 링크 | `/login` |
| 마이페이지 링크 | `/my` |
| 개인정보처리방침 | `/privacy` |
| 이용약관 | `/terms` |

## 스타일 노트
- 배경: `bg-zinc-50 dark:bg-black`
- 콘텐츠 폭: `max-w-md`, 패딩 `px-6`
