# 13. 궁합 보기

## 페이지 구성

| 경로 | 역할 | 파일 |
|------|------|------|
| `/compatibility` | 나 + 상대 정보 입력 | `src/app/compatibility/page.tsx` |
| `/loading-screen?type=compatibility` | 로딩 | 기존 loading-screen 확장 |
| `/compatibility/result` | 궁합 결과 | `src/app/compatibility/result/page.tsx` |

### 화면 구성

#### 1) 타이틀
- "궁합 보기" (2xl, bold)
- "두 사람의 생년월일로 궁합을 분석해드려요"

#### 2) 나의 정보
- **이름**: 텍스트 입력 (placeholder: "이름")
- **생년월일**: 연/월/일 드롭다운 3개
- **성별**: 남성/여성 선택 버튼

#### 3) 상대 정보
- 나의 정보와 동일한 구성
- 연예인, 친구, 연인 등 누구든 입력 가능

#### 4) CTA
- "궁합 보기" → `/loading-screen?type=compatibility`
- 비활성화 조건: 두 사람의 이름/생년월일/성별 중 하나라도 미입력

### 데이터 저장
- `sessionStorage.setItem('compatibilityInput', ...)` — 두 사람 정보
- 기존 결과 캐시 초기화: `compatibilityResult` 삭제

```
│   [홈으로]                      │ ← → /
└─────────────────────────────────┘
```

### 화면 구성

#### 1) 궁합 타이틀
- 검은 배경 카드
- "OOO ♥ OOO" (두 사람 이름)
- 궁합 점수 (100점 만점)
- 한 줄 궁합 요약

#### 2) 3축 궁합 카드
- **연애 궁합**: 점수 바 + 해석
- **친구 궁합**: 점수 바 + 해석
- **일 궁합**: 점수 바 + 해석
- 각 축별 점수(100점 만점) + 2~3문장 해석

#### 3) 조언
- 이 관계를 더 좋게 만들기 위한 구체적 조언

#### 4) 한 줄 정리
- 검은 배경, 공유하고 싶은 한 줄 카피

## API

- `POST /api/compatibility`
- Body:
```json
{
  "person1": { "name": "이름", "birthYear": 1995, "birthMonth": 3, "birthDay": 15, "gender": "male" },
  "person2": { "name": "이름", "birthYear": 1997, "birthMonth": 8, "birthDay": 22, "gender": "female" }
}
```
- Response:
```json
{
  "result": {
    "totalScore": 87,
    "summary": "서로의 빈 곳을 채워주는 관계",
    "love": { "score": 90, "interpretation": "..." },
    "friendship": { "score": 85, "interpretation": "..." },
    "work": { "score": 78, "interpretation": "..." },
    "advice": "...",
    "oneLiner": "..."
  }
}
```

## 활용 예시
- 친구 궁합: "나랑 베프 궁합 몇 점?"
- 연인 궁합: "우리 사주 궁합 어때?"
- 아이돌 궁합: "NCT 마크랑 나랑 궁합은?"
- 동료 궁합: "같이 일하면 잘 맞을까?"
