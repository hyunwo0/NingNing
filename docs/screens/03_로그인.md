# 03. 로그인 페이지

> 경로: `/login` | 파일: `src/app/login/page.tsx` | 렌더링: Client Component

## 역할

이메일/비밀번호 로그인과 소셜 로그인(카카오, Google)을 제공합니다. 회원가입은 별도 페이지(`/signup`)에서 처리합니다.

## 레이아웃

```
┌─────────────────────────────────┐
│ ← 홈으로                        │
│                                 │
│   내 운세일기를 만들어보세요      │ ← 설명
│                                 │
│   이메일                        │
│   ┌─────────────────────────┐   │
│   │ email@example.com        │   │
│   └─────────────────────────┘   │
│                                 │
│   비밀번호                      │
│   ┌─────────────────────────┐   │
│   │ ••••••••                 │   │
│   └─────────────────────────┘   │
│                                 │
│   ⚠ 에러 메시지 (조건부)        │
│                                 │
│   ┌─────────────────────────┐   │
│   │         로그인           │   │ ← 제출 버튼
│   └─────────────────────────┘   │
│   아직 계정이 없으신가요? 회원가입 │ ← → /signup
│                                 │
│   ──────────  또는  ──────────  │
│                                 │
│   ┌─────────────────────────┐   │
│   │  🟡  카카오로 시작하기    │   │ ← 노란 배경
│   └─────────────────────────┘   │
│   ┌─────────────────────────┐   │
│   │  G   Google로 시작하기   │   │ ← 흰 배경 + 테두리
│   └─────────────────────────┘   │
└─────────────────────────────────┘
```

## 화면 구성

### 1) 뒤로가기 링크
- "← 홈으로" → `/`

### 2) 설명 문구
- "내 운세일기를 만들어보세요"

### 3) 이메일/비밀번호 폼
- **이메일 입력**: type=email, placeholder="email@example.com"
- **비밀번호 입력**: type=password, placeholder="비밀번호 입력"
- **에러 메시지**: 빨간색 텍스트
- **제출 버튼**: h-12, rounded-xl, bg-foreground
  - 텍스트: "로그인" / "처리 중..."

### 4) 구분선
- "또는" 텍스트 + 양쪽 가로선

### 5) 소셜 로그인 버튼
- **카카오**: 노란 배경(#FEE500), 검은 텍스트, 카카오 로고 SVG
  - "카카오로 시작하기"
  - scope: `profile_nickname profile_image`
- **Google**: 흰 배경 + 테두리, Google 로고 SVG (4색)
  - "Google로 시작하기"
- 소셜 로그인 리다이렉트: `/auth/callback`

### 6) 회원가입 유도
- "아직 계정이 없으신가요? **회원가입**" → `/signup`

## 상태 관리
- `email`, `password`: 폼 입력값
- `error`: 에러 메시지
- `loading`: 제출 중 상태

## 인증 처리
- Supabase Auth 사용 (`getSupabaseBrowserClient()`)
- 로그인 성공 → `router.push('/')` + `router.refresh()`
- 에러 한글화: "Invalid login credentials" → "이메일 또는 비밀번호가 올바르지 않습니다"

## 이동 경로

| 요소 | 이동 대상 |
|------|-----------|
| ← 홈으로 | `/` |
| 로그인 성공 | `/` |
| 소셜 로그인 | `/auth/callback` → `/` |
| 회원가입 링크 | `/signup` |
