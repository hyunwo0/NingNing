# 03-1. 회원가입 페이지

> 경로: `/signup` | 파일: `src/app/signup/page.tsx` | 렌더링: Client Component

## 역할

이메일 인증 코드 기반 회원가입을 처리합니다. `/login`에서 "회원가입" 링크를 통해 진입합니다.

## 화면 구성

### 1) 헤더
- "← 뒤로가기" → `router.back()`

### 2) 이메일 입력 + OTP 코드 발송
- **이메일 입력**: type=email, placeholder="email@example.com"
- **확인 버튼**: 이메일 입력 우측에 배치
  - 클릭 시 Supabase `signUp` → 이메일로 6자리 OTP 코드 발송
  - 이미 가입된 이메일이면 "이미 가입된 이메일입니다" 에러
  - 발송 중: "발송 중..." (비활성화)
  - 발송 완료: "재발송" 으로 텍스트 전환

### 3) 인증 코드 입력 + 검증
- **코드 입력**: 이메일 확인 후 노출, maxLength=6
- **확인 버튼**: 코드 입력 우측에 배치
- Supabase `verifyOtp({ email, token, type: 'email' })` 으로 검증
- **결과 메시지**:
  - 성공: 초록색 "인증되었습니다"
  - 실패: 빨간색 에러 이유 표시 (예: "인증 코드가 올바르지 않거나 만료되었습니다")

### 4) 닉네임 입력
- **닉네임 입력**: placeholder="2자 이상 입력하세요", maxLength=20
- 중복 체크 없음 (user_metadata에 저장, 표시용)

### 5) 비밀번호 입력
- **비밀번호**: type=password, placeholder="8자 이상 입력하세요", minLength=8
- **비밀번호 확인**: type=password, placeholder="비밀번호를 다시 입력하세요"
- **불일치 메시지**: 빨간색 "비밀번호가 일치하지 않습니다" (입력 중 실시간 비교)

### 6) 이용약관 동의
- **이용약관 동의** (필수): 체크박스 + "이용약관에 동의합니다" + 보기 링크 → 모달로 약관 내용 표시
- **개인정보 처리방침 동의** (필수): 체크박스 + "개인정보 처리방침에 동의합니다" + 보기 링크 → 모달로 방침 내용 표시
- 모달: 반투명 오버레이 + 스크롤 가능한 본문 + "확인" 닫기 버튼

### 7) 회원가입 버튼
- **텍스트**: "회원가입" / "처리 중..."
- **스타일**: h-12, w-full, rounded-xl, bg-foreground
- **비활성화 조건**: 이메일 미인증 / 닉네임 2자 미만 / 비밀번호 미입력 / 비밀번호 불일치 / 약관 미동의

## 가입 흐름 (Supabase)
1. `signUp({ email, password: 임시 })` → OTP 코드 이메일 발송
2. `verifyOtp({ email, token, type: 'email' })` → 인증 완료 (세션 생성)
3. `updateUser({ password, data: { name: 닉네임 } })` → 비밀번호 + 닉네임을 user_metadata에 저장

> Supabase 대시보드에서 Authentication → Email Templates → Confirm signup 을
> OTP(6자리 코드) 형식으로 설정해야 합니다.

## 닉네임 저장 방식
- DB 테이블이 아닌 Supabase Auth의 `user_metadata.name`에 저장
- 소셜 로그인(카카오/Google)은 자동으로 `user_metadata.name` 또는 `full_name`이 설정됨
- 마이페이지에서 `user.user_metadata?.full_name || user.user_metadata?.name`으로 표시

## 상태 관리
- `email`: 이메일 입력값
- `verificationCode`: 인증 코드 입력값
- `emailSent`: 코드 발송 완료 여부
- `emailVerified`: 이메일 인증 완료 여부
- `verificationError`: 인증 에러 메시지
- `nickname`: 닉네임 입력값
- `password`: 비밀번호
- `passwordConfirm`: 비밀번호 확인
- `agreeTerms`: 이용약관 동의 여부
- `agreePrivacy`: 개인정보 처리방침 동의 여부
- `loading`: 제출 중 상태

## 이동 경로

| 요소 | 이동 대상 |
|------|-----------|
| ← 뒤로가기 | 이전 페이지 (router.back) |
| 이용약관 보기 | 모달 (페이지 이동 없음) |
| 개인정보 처리방침 보기 | 모달 (페이지 이동 없음) |
| 가입 성공 | 자동 로그인 후 `/` |
