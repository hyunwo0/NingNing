## 공유
- **MBTI 공유 카드**: AI 이미지 + MBTI 유형(크게) + title
- 모바일: OS 공유 시트 / PC: 이미지 다운로드

## API

- `POST /api/mbti` — MBTI 유형 → AI 운세
- Body: `{ mbtiType: string }`
- Response: `{ title, mood, advice, caution, oneLiner }`
