# NingNing 사이트맵

> 최종 갱신: 2026-04-05

## 전체 구조

```
/                          ← 메인 랜딩 페이지
├── /landing/[theme]       ← 테마별 랜딩 (love, career)
├── /login                 ← 로그인
├── /signup                ← 회원가입
├── /input                 ← 생년월일시 입력
├── /loading?type=...      ← 로딩 (사주 계산 / 심층 리포트)
├── /result                ← 운세 결과
│   ├── /question          ← AI 추가질문 (채팅)
│   └── /report            ← 심층 리포트
├── /tarot                 ← 타로 (질문 선택 + 카드 뽑기)
│   └── /tarot/result      ← 타로 결과
├── /mbti                  ← MBTI 운세 (유형 선택)
│   └── /mbti/result       ← MBTI 운세 결과
├── /compatibility         ← 궁합 보기 (두 사람 생년월일 입력)
│   └── /compatibility/result ← 궁합 결과
├── /face                  ← AI 관상 (사진 업로드)
│   └── /face/result       ← 관상 결과
├── /my                    ← 마이페이지 (로그인 필요)
├── /terms                 ← 이용약관
└── /privacy               ← 개인정보처리방침
```

## 사용자 흐름 (User Flow)

```
[신규 사용자]
  메인(/) → 입력(/input) → 결과(/result) → 추가질문(/question) 또는 리포트(/report)

[테마 유입]
  랜딩(/landing/love) → 입력(/input) → 결과(/result)

[회원가입]
  메인(/) → 로그인(/login) → 회원가입(/signup) → 자동 로그인 → 메인(/)

[회원 사용자]
  메인(/) → 로그인(/login) → 메인(/) → 입력(/input) → 결과(/result) → 마이페이지(/my)
```

## 공통 레이아웃

- **RootLayout** (`layout.tsx`): Geist 폰트, AuthProvider 래핑, `lang="ko"`, Toaster
- **GNB** (`components/layout/GNB.tsx`): 모든 페이지 상단에 적용 → [00-1_GNB.md](./00-1_GNB.md)
  - 메인: 좌측 NingNing 로고(비활성) + 우측 로그인/마이페이지
  - 나머지: 좌측 ← 뒤로가기 + 우측 로그인/마이페이지
- **배경**: `bg-zinc-50 dark:bg-black`
- **콘텐츠 폭**: `max-w-md` (모바일 퍼스트)
- **다크모드**: 시스템 설정 자동 반영

  tarotInput         ← /tarot에서 저장 → /loading-screen?type=tarot에서 사용

  mbtiResult         ← /loading-screen에서 저장 → /mbti/result에서 캐시
