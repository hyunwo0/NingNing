# 운세 타입 확장 계획

> 작성일: 2026-04-06

## 목표

현재 사주 기반 오늘의 운세 외에 다양한 운세/콘텐츠 타입을 추가하여 사용자 재방문과 체류 시간을 늘린다.

## 추가 예정 타입

### 1. 타로
- 사용자가 카드를 선택하는 인터랙션
- 선택한 카드 조합을 AI가 해석
- 질문 유형: 연애, 진로, 오늘의 한마디 등

### 2. 관상
- 사용자가 얼굴 사진을 업로드
- AI가 관상학 기반으로 성격/운세 분석
- Vision API 활용 가능

### 3. MBTI 운세
- MBTI 유형 선택 또는 간단 테스트
- MBTI + 사주 조합으로 오늘의 운세/조언
- 사주와 MBTI의 교차 분석

### 4. 나와 맞는 연예인
- 사주/MBTI/성격 기반으로 궁합이 맞는 연예인 매칭
- 공유하기 좋은 바이럴 콘텐츠
- 연예인 데이터 DB 필요

## 검토 사항

- [ ] 각 타입별 입력 화면 설계 (공통 컴포넌트 재활용 가능한지)
- [ ] 결과 화면을 공통으로 쓸 수 있는지 vs 타입별 전용 화면
- [ ] 로딩 페이지 문구 타입 추가 (기존 `/loading-screen` 확장)
- [ ] 메인 페이지에서 타입 선택 UI (탭? 카드? 스와이프?)
- [ ] 무료/유료 정책 (타입별 차등?)
- [ ] 관상: 이미지 업로드 → Vision API 비용/개인정보 이슈
- [ ] 연예인 매칭: 연예인 사주/MBTI 데이터 수집 방법

## 우선순위 (제안)

| 순위 | 타입 | 이유 |
|------|------|------|
| 1 | MBTI 운세 | 입력 간단, 사주와 시너지, 바이럴 가능 |
| 2 | 타로 | 인터랙션 재미, 사주 없이도 독립 서비스 가능 |
| 3 | 나와 맞는 연예인 | 바이럴 최강, 데이터 준비 필요 |
| 4 | 관상 | Vision API 비용 + 개인정보 리스크 |
