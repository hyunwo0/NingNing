# 운세 톤앤매너 리빌딩 계획

> 작성일: 2026-04-07

## 문제

현재 운세 결과가 "전통 한국식 사주 용어" 중심이라 글로벌 어필이 약함.
- "식상(食傷)의 기운이 활발해..." → 무슨 말인지 모름
- "관성이 강해지는 시기" → 전문가가 아니면 접근 불가
- 전체적으로 점술사 느낌 → 트렌디하지 않음

## 방향

**"K-감성 + 직관적 + 밈화 가능한 구조"**

- 사주 용어는 백그라운드로 숨기고, 프론트는 감각적 표현
- 영어 키워드를 섞어 K-콘텐츠 감성 연출
- 공유하고 싶어지는 한 줄 카피
- 이모지 활용으로 시각적 리듬감

## 결과 화면 리빌딩 구조

### Before (현재)

```
한 줄 총평: "오늘은 감정 표현이 자연스럽게 통하는 하루입니다"
키워드: [소통] [표현력] [흐름]

연애: 식상의 기운이 활발해 평소보다 말이 잘 통하는 흐름이에요.
일/직장: 관성이 안정적이라 기존 업무를 정리하기 좋습니다.
재물: 재성이 약하므로 큰 지출은 피하세요.
```

### After (리빌딩)

```
🌗 Today's Core Mood
"Hidden Power Mode"
→ 힘은 충분한데, 지금은 굳이 드러낼 필요 없는 날

💖 LOVE ENERGY — Soft Balance Mode
감정은 안정적. 깊은 관계보다 "편안한 연결"이 좋은 날
👉 Tip: 과한 표현보다 센스 있는 한마디

💼 WORK / LIFE — Clean Focus Mode
일 잘 풀리는 날 (특히 정리 & 구조화)
👉 키워드: "정리 = 생산성"

💰 MONEY FLOW — Hold & Maintain
큰 수익보다 지키는 흐름. 충동 소비만 피하면 안정적
👉 Tip: 오늘은 "지출 컨트롤"이 이득

🎯 TODAY'S STRATEGY
"Don't flex. Just stack."
(보여주기보다 쌓는 날)
```

## 새로 추가할 섹션

### 🍀 LUCK BOOSTERS (K-STYLE PICK)

기존 "행운 단서"를 확장해서 구체적이고 실행 가능한 형태로.

| 항목 | 예시 |
|------|------|
| ✨ Style Code | Silver / White 아이템, 미니멀 악세사리 |
| 🔢 Lucky Move | "6" vibe — 선택 고민될 때 6번 옵션 |
| 🧭 Energy Direction | North = Water Energy Zone |
| 🕒 Golden Time | 15:00 – 17:00 — 중요한 대화/결정은 이 시간에 |

### 🌊 K-ORACLE INTERPRETATION

사주 근거를 글로벌 감성으로 번역하는 섹션.

```
Before: "오늘은 금(金)의 기운이 강한 날입니다"
After:  "Structure + Precision + Control Energy Day"
        감정보다 판단이 앞서는 날. 빠르게보다 "정확하게" 가는 날
```

### 💬 한 줄 정리

공유하기 좋은 한 줄 카피. 밈화 가능한 포맷.

```
"오늘은 티 안 나게 잘하는 사람이 승자다."
"Don't flex. Just stack."
```

## 톤 가이드

### DO
- 영어 키워드 믹스 (Mode, Status, Tip, Energy, Vibe)
- 이모지로 섹션 구분 (💖 💼 💰 🍀 🎯)
- 짧은 문장 + 화살표(→)로 풀어쓰기
- "~하는 날" 패턴으로 직관적 전달
- 공유하고 싶은 한 줄 카피 (영어 + 한국어 병기)

### DON'T
- 사주 전문 용어를 프론트에 노출 (식상, 관성, 재성, 인성 등)
- "~합니다" 체의 딱딱한 존댓말
- 공포 조장, 확정적 예언
- 길고 장황한 해석
